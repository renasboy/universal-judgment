# universal-judgment
Universal Judgement (www.universaljudgment.com)

# Backend

- python3-venv
- python3-pip


# Angular build

- node.js (https://nodejs.org/en/)
- npm insatall
- npm install bower
- npm install gulp
- bower install


# Angular gulp run

- Build dist: gulp build:dist
- Serve: gulp serve (on port 3000)
- Serve dist: gulp serve:dist
- Remove dist folder: gulp clean
